# wuwa-cubie-derby

This repository is used for simulating the cubes in Wuthering Wave Cubie Derby event to know the likelihood of a Cube winning.

## Running the script

1.  Clone the repository

2.  Install dependencies
    ```bash
    pip install -r requirements.txt
    ```

3.  Run `game.py`
    ```bash
    python game.py
    ```
